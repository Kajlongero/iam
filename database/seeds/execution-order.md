# Instructions

- Files are marked with: **xx.filename.extension**

Legend:

- xx: File number
- filename: File name
- extension: Extension of the file

i.e 01.index.sql

You just need to follow the order of the file number in asc form

## Execution Priority

Follow the order of numbers in the files in asc order, **COPY FILES AND EXECUTE IN ORDER**
